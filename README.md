# linkedin-or-interpol-solver
a funny script i made for the game https://linkedin-or-interpol.com/

# how to use
it will show a green dot in the bottom left if the answer is LinkedIn, and red if the answer is Interpol.

# how it works
first we start of by injecting the dot css into the DOM, then we intercept the api and capture the response from /api/random-person endpoint.
